verifyWebhookSignature()

Client.php : 333

Verify a webhook signature using the configured webhook secret

public verifyWebhookSignature(string|StreamInterface $body, string $signature[, string $algorithm = 'sha256' ]) : void

Convenience method that automatically uses the webhook secret from the Config object. Throws an exception if verification fails.

Example usage:

try {
    $client->verifyWebhookSignature(
        file_get_contents('php://input'),
        $_SERVER['HTTP_X_SIGNATURE'] ?? ''
    );
    // Signature is valid - process webhook
} catch (WebhookVerificationException $e) {
    http_response_code(401);
}
Parameters
$body : string|StreamInterface

The webhook request body

$signature : string

The signature from webhook header

$algorithm : string = 'sha256'

The hash algorithm to use (default: sha256)

Tags
throws
WebhookVerificationException

If verification fails or secret is not configured
